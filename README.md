# Oluşturulan Türkçe Veri Setiyle Geniş Dil Modeli Tabanlı Sahte Haber Tespiti Uygulaması: Doğrulamaç
TÜBİTAK 2204-A Lise Öğrencileri Araştırma Projeleri Yarışması kapsamında hazırlanan _Oluşturulan Türkçe Veri Setiyle Geniş Dil Modeli Tabanlı Sahte Haber Tespiti Uygulaması: Doğrulamaç_ projenin reposudur.
